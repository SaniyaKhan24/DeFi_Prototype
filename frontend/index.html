Easiest path (pick one)
  1. Deploy: npm run migrate:gui · export: npm run export:frontend.
  2. Run npm run dev (recommended). Open the URL it prints (default http://127.0.0.1:8788/frontend/) — includes /rpc-proxy for Ganache-key mode.
  3. MetaMask: add network manually (see Setup help). Import Ganache mnemonic so you have ETH. Then Connect wallet.
  4. No MetaMask? Select Ganache private key, paste a key from Ganache → Accounts → key icon, then Connect wallet (only works with npm run dev).
  5. Paste DecentralizedBank contract address from migrate output — not a Ganache account address.
