MetaMask's "add network" buttons are unreliable (chain id vs network id). Add the network by hand once:

  1. MetaMask -> Networks -> Add network -> Add a network manually.
  2. New RPC URL (Ganache UI default): http://127.0.0.1:7545
  3. Chain ID: try 1337 first. If MetaMask complains, use 5777.
  4. Currency symbol: ETH | Chain name: Local Ganache (any label).

CLI Ganache on port 8545? Use RPC http://127.0.0.1:8545 and the same chain ID shown at startup.

Then import Ganache's mnemonic (or a private key) so your account has test ETH on that network.
